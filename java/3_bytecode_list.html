以上指令可以分为如下几类，方便记忆:

# 加载指令

  • load类指令: 将局部变量表中的变量 加载到 操作数栈。针对的是变量。
  • const&push: 将常量值加载到操作数栈顶。针对的是具体常量值。
  • ldc: 从常量池加载对应下标的常量到操作数栈顶。针对的是常量池的下标。 具体可查表。

# 存储指令

  • store类指令: 将操作数栈顶数据 存储到 局部变量表。

# 操作数栈指令

  • pop指令: 将栈顶元素出栈。
  • dup指令: 复制栈顶元素并压入栈顶。
  • swap指令: 交换站定两个元素的值。

# 运算指令

  • add指令: 加法
  • sub指令: 减法
  • mul指令: 乘法
  • div指令: 除法
  • rem指令: 求余
  • neg指令: 取负
  • and指令: 逻辑与
  • or指令: 逻辑或
  • xor指令: 逻辑异或

# 类型转换指令

  • x2y: 将x转换为y

# 控制转移指令

  • ifxxx: 条件转移指令(xxx为条件，比如eq就是equal，所以ifeq就是if equal也就是:如果相等)
  • tableswitch/lookupswitch: 复合条件转移指令
  • goto/goto_w/jsr/jsr_w/ret: 无条件转移指令

# 方法调用指令

  • invokexxx: 调用xxx类型方法。
